Student Course Registration System
1. Planning

The goal of this project is to design a simple system that allows students to register courses.
The system will collect student name, matric number, and selected courses.

2. Requirements Analysis

Functional Requirements:

User can enter student name

User can enter matric number

User can select courses

System displays registered courses

Non-Functional Requirements:

Easy to use

Runs on Python

Clear output

3. System Design

The system is designed as a command-line Python application.
Main components:

Input collection

Data storage using variables

Output display

4. Implementation

The system is implemented using Python.
The program collects user input and displays registration details.

5. Testing

The program is tested by entering different student details to ensure correct output.

6. Deployment

The application is deployed by uploading it to GitHub for access and submission.

7. Maintenance

Future improvements may include:

Database integration

Graphical user interface

User authentication
